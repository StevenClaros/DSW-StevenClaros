Que es CSS y su Sintaxis

Inicialmente vamos a definir qué es CSS, este es el formato para páginas creadas en formato HTML basada en los estándares de hojas estilo cascada, su uso le permite ahorrar tiempo, dar consistencia y facilitar la escritura de páginas. Según Sirvent Collado, L., & Luján-Mora, S. (2013) la sintaxis consta de dos partes, el selector y la declaración, por ejemplo: h2 {color:black} En éste ejemplo la etiqueta h2 sería el selector, notifica que el estilo definido se aplicaría a los elementos h2 y la declaración color:black indica que el color del texto de tolos los elementos deben ser de color negro. La declaración se debe encerrar entre llaves y contener las definiciones. Cada una consta de propiedad/valor, donde se separa por dos puntos selector {propiedad:valor} El selector es el nombre de un elemento HTML pero también puede ser de clase, un ID de selector o un selector de contexto, si el valor tiene más de una palabra se pone entre comillas. Para aplicar estilos a varios elementos se pueden agrupar los selectores que comparten definiciones. Para ello se escribe un elemento tras otro separado por coma.



La sintaxis está compuesta por sentencias en las que se establecen atributos de estilo y valor para uuna propiedad, éstas se separan por dos puntos.

A éstas sentencias se le aplicar a las etiquetas de HATML estableciendo características para su aspecto como terminología, alineación, color o tamaño
